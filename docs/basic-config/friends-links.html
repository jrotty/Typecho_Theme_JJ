友情链接页 ​

搭建一个友情链接页 ​

点击博客的独立页面菜单 -> 新增一个独立页 -> 内容如下：

标题：友链

名字：links

页面顺序：2

模板：links

公开度：公开

权限控制：全选

填好后发布页面即可。
