# 主题构建

如果你是一个前端开发，且有 SPA 单页面开发的经验，并且有自己的个性化定制需求，那你可以继续往下阅读，如果不是，下面的内容可能会让你头疼。

## 环境

- Node >= v18.15.0
- pnpm >= 7.29.2

## 命令

**安装依赖**

```bash
pnpm i
```

**开发环境启动**

```bash
pnpm run dev
```

**正式打包**

```bash
pnpm run build
```

**包体分析**

```bash
pnpm run analyze
```
