# 网站地图

这是一个用于给爬虫抓取网页信息的功能，通过插件实现！

## 插件下载

点击下载：[AutoSitemap](/zip/AutoSitemap.zip)

## 安装

下载完丢入 Typecho 的 plugins 目录下，解压，然后去博客后台启用该插件即可。
